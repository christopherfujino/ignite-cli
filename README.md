# ignite-cli

Simple CLI interface for Ignite; scaffold and setup your Flame projects with ease.

## Instructions

Run

```bash
dart test
```

To run all tests. Always make sure the build is green.

In order to test the application, run

```bash
dart dart bin/ignite_cli.dart
```
